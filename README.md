## 🛠 set-up

1. Install the dependencies

   ```sh
   npm install or yarn
   ```

2. Start the development server

   ```sh
   yarn run start
   ```

## 🚀 build and run for production

1. Generate a full static production build

   ```sh
   npm run build
   ```
